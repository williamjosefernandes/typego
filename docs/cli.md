# CLI

## Commands

- `typego create <name>`: create a new project from templates
- `typego new <name>`: alias for create
- `typego build`: parse TypeScript controllers and emit Go source
- `typego dev`: build and watch `src` files
- `typego start`: helper command for runtime hints
- `typego generate <kind> <name>`: scaffold files (`controller`, `service`, `module`)
